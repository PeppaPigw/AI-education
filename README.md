# AI-Education

## Usage

### Env

For python==3.12,you can configure the environment using the following command

```bash
pip install -r requirements.txt
```

### Run

- For Gradio

```python
python AI-Education/main.py
```

open on localhost

- For FastAPI

```python
python main-fastapi.py
```

open on http://localhost:8000/
